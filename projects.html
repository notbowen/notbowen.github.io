My Projects

SimpleOS

Description:
SimpleOS is a bare metal operating system that I wrote in hopes of gaining a better understanding of how computers and operating systems worked. I followed this tutorial, along with this book and the OS Dev Wiki.

OS Features:

  • Printing to screen
  • Snake Game
  • 4 simple shell commands, echo, cls, end and snake
  • Echo accepts variable length arguments
  • Time.sleep using the PIT, pseudo-random numbers using linear congruential generators

Stuff I learnt:
  • Simple assembly commands, mov, push, pop
  • Logic in assembly: cmp, je, jne
  • Functions in assembly: loading arguments into registers, jmp and ret
  • Bootsector and BIOS functions: e.g. printing to screen and reading from disk
  • Interrupts, Programmable Interval Timers, Keyboard inputs
