My Projects

SimpleOS

Description:
SimpleOS is a bare metal operating system that I wrote in hopes of gaining a better understanding of how computers and operating systems worked. I followed this tutorial, along with this book and the OS Dev Wiki.

OS Features:

  • Printing to screen
  • Snake Game
  • 4 simple shell commands, echo, cls, end and snake
  • Echo accepts variable length arguments
  • Time.sleep using the PIT, pseudo-random numbers using linear congruential generators

Stuff I Learnt:
  • Simple assembly commands, mov, push, pop
  • Logic in assembly: cmp, je, jne
  • Functions in assembly: loading arguments into registers, jmp and ret
  • Bootsector and BIOS functions: e.g. printing to screen and reading from disk
  • Interrupts, Programmable Interval Timers, Keyboard inputs

Discord Bot Autofarm

Description:
I came across an article praising Python on how easy it was to automate stuff with Python. Naturally, I got curious and started to look more into automation with Python. The first version of this project made use of Selenium to log into Discord, before typing into Discord's text box and sending. However, this version would easily be messed up when a user typed into the text box along with the bot. I then discovered that Discord had an API that made use of a user's token to send messages into channels. I thus made use of this API and rewrote the entire project, learning a lot of new stuff along the way.

Features:

  • User preferences and saving
  • Error logging and handling
  • Coloured output :D

Stuff I Learnt:
  • Multithreading and Function Scheduling
  • Printing color to the terminal
  • Overwriting previously printed stuff
